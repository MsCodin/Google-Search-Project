# Google Search Project

A front-end clone of Google Search built with HTML and CSS.

## Usage

Open `index.html` in a browser, or use a local server such as VS Code Live Server.

Navigate between three pages using the links in the upper-right corner of each page:

- **index.html** — Google Search with "I'm Feeling Lucky"
- **GoogleImage.html** — Google Image Search
- **GoogleAdvanced.html** — Google Advanced Search with filters for all words, exact phrase, any words, and excluded words
